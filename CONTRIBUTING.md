# Contributing to Graph Congestion Lab

First off, thanks for taking the time to contribute! 🎉

## Development Setup

1. Fork the repo and clone it locally.
2. Install dependencies: `pip install -r requirements.txt`.
3. Ensure you have `ruff` and `pytest` installed for local testing.

## Submitting Pull Requests

1. Create a new branch: `git checkout -b feature/your-feature-name`.
2. Make your changes.
3. Run the linter: `make lint` (or `ruff check .`).
4. Run the test suite: `make test` (or `pytest`).
5. Commit your changes with a descriptive message.
6. Push to your fork and submit a PR!

## Issue Reporting

If you find a bug or have a feature request, please open an issue with:
- A clear description of the problem/request.
- Steps to reproduce (if it's a bug).
- Expected behavior vs actual behavior.
